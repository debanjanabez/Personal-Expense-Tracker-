This project allows users to record and manage daily expenses. It showcases a multi-language approach where each language is responsible for a specific part of the system, and all components communicate through a shared file.

expense-tracker/
│
├── main.py          # Python: Handles input and stores data
├── Main.java        # Java: Reads and processes expense data
├── program.c        # C: Displays summaries and totals
├── expenses.txt     # Shared data file
└── README.md
